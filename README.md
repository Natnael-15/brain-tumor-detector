# Brain Tumor Detector

## Description
This project provides a comprehensive solution for brain tumor detection using advanced algorithms. The primary focus is on the Brain Activity Simulation component, which displays real-time animated brain wave monitoring (alpha, beta, gamma, delta waves), neural activity visualization, and tumor detection indicators using HTML5 Canvas.

## Features
- Real-time animated brain wave monitoring
- Neural activity visualization
- Tumor detection indicators
- Interactive user interface
